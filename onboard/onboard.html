Guide & Key Features

  • 1. Real-Time Suggestions:

    Real-time suggestions appear as you type. This feature is optimized with a 500ms debounce for speed and a 2 second throttle for cost efficiency.

  • 2. Copy/Accept Suggestion:

    Quickly copy a suggestion using the shortcut: Ctrl/Cmd + Shift + Y and then paste it in the input box.

  • 3. Tone Adaptation (Two-Tier):

    Use the popup to either set your real-time tone (Formal or Casual) or use the refinement tool to rewrite a full draft before sending.
